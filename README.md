# Phonepe_Analysis
![Logo](https://github.com/ojaswi43/Phonepe_Analysis/blob/main/Logo.webp)

# Overview 
 + Designed and developed an interactive PhonePe Transaction Analytics Dashboard to monitor transaction performance, user behaviour, payment success rates, and service adoption trends.
 + Analyzed 288K transactions worth ₹3.33 Billion across 108K unique users to uncover transaction patterns and customer demographics.
 + Created KPI cards, trend analysis, demographic segmentation, and weekday/weekend usage visualizations to support data-driven business decisions.
 + Identified peak transaction periods, high-value service categories, and user segments contributing the most to transaction volume.
